# 🗳️ Online Voting System

## 📌 Project Overview
The Online Voting System is a Java console-based application developed to simulate a basic electronic voting process. The system allows registered voters to cast their vote for a candidate while ensuring that each voter can vote only once. It also includes an admin login feature to securely view the election results.

---

## 🚀 Features
- Display list of candidates
- Cast vote using a Voter ID
- Prevent duplicate voting
- Admin login for viewing results
- Display vote count for each candidate
- Announce the winning candidate
- Simple menu-driven interface
- Easy to use and understand

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project in VS Code, Eclipse, or IntelliJ IDEA.
3. Compile the Java program:

```
javac OnlineVotingSystem.java
```

4. Run the program:

```
java OnlineVotingSystem
```

5. Select an option from the menu.
6. Enter your Voter ID to cast a vote.
7. Log in as an administrator to view the election results.
